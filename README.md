# HEARTHFALL — 創発シミュレーション プロトタイプ

「世界一のシミュレーションゲーム」を目指す設計図と、その縦スライス実装。
プレイヤーの介入（場の操作）が **生態 × 経済 × 心理** の3層を貫いてフィードバックし、
開発者が書いていない「物語」が勝手に立ち上がることを狙う。

## 中身

| ファイル | 内容 |
|---|---|
| `index.html` | 動作するプロトタイプ本体（ビルド不要・単一ファイル） |
| `docs/simulation-game-design.md` | 完全な設計図（コンセプト/複雑系設計/技術アーキテクチャ/UI哲学） |

## 遊び方（起動）

ビルド不要。ブラウザで `index.html` を開くだけ。

```bash
# どれか1つ
open index.html                 # macOS
xdg-open index.html             # Linux
python3 -m http.server 8000     # その後 http://localhost:8000 を開く
```

- **オーバーレイ**（右パネル）で 植生 / 肥沃度 / 水分 / 汚染 の場を可視化
- **住人をクリック** すると、感情の内訳（因果追跡）・性格(OCEAN)・関係・記憶を確認できる
- **📜 年代記** に、絆の成立・出産・諍い・長老の死などの創発イベントが綴られていく
- 速度切替（x1/x3/x8）・一時停止・「新しい世界」で別シードの世界を生成

## 実装されているシステム（縦スライス）

- **生態層**: 拡散方程式（離散ラプラシアン）による水分・汚染の伝播、季節変動、植生の成長と疲弊
- **心理層**: PAD感情モデル / Utility AI による行動決定 / Big Five 性格 / 重み付き関係グラフ /
  忘却曲線つき記憶
- **創発ループ**: 食料の採集・消費、絆の成立、出産による世代交代、加齢と寿命、
  死を看取った住人への記憶（悲嘆）、年代記による物語化
- **基盤**: ECS風のデータ指向構成、シード付きPRNGによる決定論的シミュレーション

## 検証

`node` でヘッドレス実行し、長期シミュレーションが例外なく安定し、
創発イベント（絆・出産・世代交代）が生成されることを確認済み。
直近の200日検証では人口が初期12人から自律的に増減し均衡、例外0で生成イベントが継続。
